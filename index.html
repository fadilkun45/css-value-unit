Apa itu css unit

css unit adalah sebuah satuan ukuran yang di gunakan dalam css. biasa nya kita hanya mengetahui px dan %, namun ini beda lagi ini akan lebih kompleks lagi dan juga lebih membuat web jadi lebih mudah untuk responsive.

EM

EM adalah satuan unit css yang mengambil dari ukuran font size parent class nya. ukuran 1EM adalah sebesar 1kali ukuran parent class nya dan 2EM berati 2 kali ukuran font parent class nya. bila parent class pertama nya tidak menpunyai ukuran font maka si EM akan terus mengambil hingga ke root. contoh

REM

REM adalah satuan unit css yang mengambil ukuran font size dari html atau gk root dan juga dia tidak mengambil dari parent class nya. sama seperti EM 1 REM sama dengan 1 kali ukuran font size si html dan juga root jika 2REM berati 2 kali nya dan seterus nya. contoh
